# Content_Management

Tech stack: React - GoLang - MongoDB

Dependencies:
Go 1.14
Node v12.10

Start the Server:
1) cd server
2) go run main.go

Start the Client:
1) cd client
2) npm start

Puppeteer FB post:
1) In root directory run:
1) npm i puppeteer
2) npm i axios
3) node pups.js 

Cron Jon running Puppeteer FB post:
1) In root directory run:
2) npm install express node-cron
3) node cronJob.js
